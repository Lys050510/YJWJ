## [2026-07-19] — 修复全新环境初始化 Bug

### 修复
- **修复全新环境首次访问全部功能失效的严重 Bug**
  - 根因：`config.js` 初始化时，空 localStorage 被设为 `{configVersion:0}`（truthy），导致三元表达式错误地取到 `undefined`，`activeHeroNames` 等 4 个字段变为 `undefined` 并持久化到 localStorage
  - 影响范围：新浏览器、无痕窗口、手机端首次访问 — 英雄/武器抽取按钮无反应、人员头像不显示、计分板英雄头像不显示
  - 桌面端已访问过的浏览器不受影响（localStorage 已有正确缓存走 else 分支）
  - `configVersion` 升级至 16，自动修复已损坏的缓存
  - 为迁移分支和 else 分支补充 9 处缺失字段的兜底检查

## [2026-07-19] — 正式上线 & 性能优化

### 新增
- 网站正式上线，域名 **[yjwj666.top](https://yjwj666.top/)**
- Cloudflare CDN 加速，国内用户可直接访问
- 新增 `CHANGELOG.md` 更新日志

### 优化
- **加载速度大幅提升**：首次加载体积从 ~60MB 降至 ~1.5MB
  - 12 个 CSS 文件合并为 1 个，减少 11 次 HTTP 请求
  - 全部图片转为 WebP 格式（卡片背景 50MB → 0.6MB）
  - JS 模块预加载，并行下载
- 新增图片压缩脚本 `python tools/compress_images.py`

### 修复
- 修复全部 XSS 隐患（8 处），用户输入统一转义
- 修复 `localStorage` 同源冲突（键名基于路径命名空间）
- 修复 `localStorage` 无保护读写，新增 try/catch + 容量预警
- 修复 `<img>` 加载失败时的脆弱的 DOM 查找，改用 `querySelector`
- 修复配置版本迁移逻辑（旧数据兼容）
- 日志自动裁剪：奖品 ≤500 条，锦囊 ≤200 条
